# Todos Server

> Todo application with Node, Sequelize and Postgres running in Docker containers.

This app performs the 5 HTTP methods (GET, POST, PUT, PATCH, DELETE). All end-points displayed in root url.

## Todo Model:
| Attribute    |  Type  |  
|------------|-------|
|    Title     | STRING |
|  Description |  TEXT  |
|    isDone    |  BOOL  |
